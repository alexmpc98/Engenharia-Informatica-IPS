Detailed Description

Defines the types MapKey and MapValue.

The MapKey is an alias to the type of keys held by an instance of the ADT Map (PtMap).

The MapValue is an alias to the type of values held by an instance of the ADT Map (PtMap).

These aliases just be changed according to the use-case.
